Class ProxyFS::Mirror
In: lib/mirror.rb
Parent: ActiveRecord::Base

Represents a remote Mirror and provides the relevant remote file system operations. Uses ActiveRecord for persistency. 

  mirror = Mirror.create :hostname => "example.com", :username => "username", :base_path => "/path/to/destination"
  mirror.mkdir "/test"
  mirror.write_to("/test/test.txt", "test")
  # ...

If unsuccessfull, the remote operations raise exceptions. The exception is one of Timeout::Error, Net::SFTP::StatusException or Errno::…
