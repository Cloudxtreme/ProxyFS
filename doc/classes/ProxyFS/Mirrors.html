Class ProxyFS::Mirrors
In: lib/mirrors.rb
Parent: Object

Manages the collection of remote mirrors to appear like if we have to communicate with a single mirror. E.g., to create a replication Task (mkdir) for every mirror, you have to call only once: 

  Mirrors.instance.mkdir("/new/directory") do
    # create directory on local mirror here (i.e. the local filesystem)
  end
